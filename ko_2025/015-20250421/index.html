📱 해상도 단위 정리

플러터로 개발하면서 안드로이드와 iOS간의 UI 차이가 OS 단위에서 차이인지 확인하기 위해 OS별 해상도에 대해서 간단히 살펴봤습니다.

  • 해상도 : 화면에 표시되는 픽셀의 총 갯수 입니다.
  • 픽셀 밀도(ppi, pixels per inch) : 화면의 선명도를 결정하는 중요한 요소로, 1인치 안에 들어가는 픽셀의 수
  • 기기 픽셀 비율 (Device Pixel Ratio, DPR)
